# Tech Stack

- **Frontend Framework:** React 19
- **Language:** TypeScript
- **Build Tool:** Vite
- **Deployment & Hosting:** Cloudflare Pages
- **Serverless API:** Cloudflare Pages Functions (`functions/api/`)
- **Database:** Cloudflare D1 (SQLite)
- **Object Storage:** Cloudflare R2
- **Styling:** Vanilla CSS (`styles.css` / `admin.css`)
- **State Management:** React Context API / Local Storage for client state
